# Widget Screenshot Placeholders

This directory contains placeholder assets for WhatsNewKit widget showcase.
Replace these placeholders with actual widget screenshots:

## Required Screenshots (PNG format, @2x and @3x):

### Lock Screen Widgets:
1. **lock-screen-circular.png** - Circular progress widget on lock screen
2. **lock-screen-rectangular.png** - Rectangular widget with building icon and progress

### Home Screen Widgets:
3. **home-screen-medium.png** - Medium (4x2) widget with full interface

## Screenshot Guidelines:
- Use iPhone 16 Pro simulator for consistency
- Capture widgets in both light and dark mode if needed
- Ensure building.2.fill and clock.badge.fill icons are visible
- Show realistic progress data (e.g., 8 of 12 days)

## File Naming Convention:
- Base: `widget-name.png`
- Retina: `widget-name@2x.png`  
- Super Retina: `widget-name@3x.png`
